<b>D acu-t usenfar n Common Voice?</b>

Common Voice d assumer n Mozilla i tallalt n uselmed n tmacinin ɣef wamek i ttmeslayen yimdanen n tidet.
Taɣect d tagmawit, taɣect n umdan. Ɣef waya i nḥar ad d-nesnulfu tatiknulujit n taɣect i zemrent ad sqedcent tmacinin-nneɣ. Maca i usnulfu n yinagrawen n taɣect, ineflayen sran tasmekta meqqren n yisefka n taɣect.
Amur ameqqran n yisefka i seqdacent tkebbaniyin timeqqranin ur yewjid ara i tugget n medden. Nettwali aya yettarez asnulfu. Ihi nsekker-d Common Voice, asenfar ara yerren aɛqal n taɣect yeldi i yal yiwen akken ad ikcem ɣur-s.
Tura tzemreḍ ad d-tefkeḍ taɣect-ik•im akken ad aɣ-tɛawneḍ ad d-nebnu taffa n yisefka n taɣect n uɣbalu yeldin i yezmer ad isseqdec yal yiwen deg usnulfu n yisnasen i yibenken akked uẓeṭṭa web. Ɣeṛ tafyirt akken ad tɛawneḍ timacinin ad lemdent amek i ttmeslayen yimdanen n tidet. Senqed amahil n yimettekkiyen-nniḍen i usnerni n tɣara. Ishel maḍi!
Tzemreḍ ad tafeḍ ugar n telqayt deg usmel web unṣib https://commonvoice.mozilla.org/

<p>
<b>Ayɣer i ilaq ad tesqedceḍ asnas deg wadeg n usmel web ɣef uziraz?</b>

Asnas yettusnefli swaswa i tiliɣriyin tuḥricin ihi d anaṣli. Llant ddeqs n tmahilin ur d-yettak ara usmel web, am usentel aberkan, askar aruqqin, aggan, iwehhiyen, atg.
Asnas fessus ugar n usmel web, war imrayen ur nemɛin (imrayen deg usnas tzemreḍ ad ten-tesḥebseḍ deg yiɣewwaṛen).
Rnu ɣer-s, asmel web yesɛa ddeqs n yibugen, ladɣa deg usmel aziraz, ihi ur tzemmreḍ ara ad tgeḍ ula d kra akken iwata.
