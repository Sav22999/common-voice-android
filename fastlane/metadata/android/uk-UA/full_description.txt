<b>Що за проект Common Voice?</b>

Загальний Голос - це ініціатива Мозіли допомогти навчати машин, як говорять справжні люди.
Голос природний, голос - це людина. Тому нам до вподоби створювати сприятливі голосові технології для нашої техніки. Але для створення голосових систем розробникам необхідна надзвичайно велика кількість голосових даних.
Більшість даних, що використовуються великими компаніями, не доступні для більшості людей. Ми вважаємо, що це стримує інновації. Ми запустили Загальний Голос, проєкт, щоб допомогти зробити розпізнавання голосу відкритим і доступним для всіх.
Тепер ви можете пожертвувати свій голос, щоб допомогти нам створити голосову базу даних з відкритим вихідним кодом, яку кожен зможе використовувати для створення інноваційних програм для пристроїв та мереж. Прочитайте речення, щоб допомогти машинам зрозуміти, як говорять справжні люди. Перевірте роботу інших учасників для покращення якості. Це ж так просто!
You can find more details on the official website <a href="https://commonvoice.mozilla.org">https://commonvoice.mozilla.org</a>
</p>

<b>Чому ви повинні використовувати додаток замість веб-сайту в телефоні?</b>

Додаток розроблений спеціально для смартфонів. Містить багато функцій, які веб-сайт не надає вам, наприклад, темну тему, режим оффлайн, жести, і т.д.
Додаток більш легкий за сайт, без набридливої анімації (анімація у додатку також може бути зупинена в налаштуваннях).
Крім того, сайт має багато помилок, особливо для мобільної версії, тому все не може бути правильно.
