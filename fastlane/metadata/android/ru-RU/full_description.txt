- Пользовательский интерфейс (UI) продуманный, понятный и современный

- Поддерживается темная тема
- Дневная цель
- Жесты
- Сохранить записи на устройстве
- Множество других возможностей
- Приложение полностью настраивается в настройках

<b>Что такое Common Voice?</b>

Common Voice - это инициатива Mozilla, направленная на обучение машин тому, как говорят реальные люди.
Голос естественный, голос человеческий. Вот почему мы с нетерпением ждем возможности создать полезную голосовую технологию для наших машин. Но для создания голосовых систем разработчикам требуется чрезвычайно большой объем голосовых данных.
Большая часть данных, используемых крупными компаниями, недоступна большинству людей. Мы считаем, что это сдерживает инновации. Поэтому мы запустили проект "Common Voice", призванный помочь сделать распознавание голоса открытым и доступным для всех.
Теперь вы можете пожертвовать свой голос, чтобы помочь нам создать базу данных голосов с открытым исходным кодом, которую каждый может использовать для создания инновационных приложений для устройств и интернета. Прочитайте предложение, чтобы помочь машинам узнать, как говорят настоящие люди. Проверяйте работу других авторов для улучшения качества. Это так просто!
Более подробную информацию вы можете найти на официальном сайте <a href="https://commonvoice.mozilla.org/">https://commonvoice.mozilla.org/</a>
</p>
<p>
<b>Почему вы должны использовать приложение вместо веб-сайта на мобильных устройствах?</b>

Приложение разработано специально для смартфонов, поэтому оно является нативным. В нем есть множество функций, которые не предоставляют веб-сайты, например, темная тема, автономный режим, персонализация, жесты и т.д.
Приложение легче, чем сайт, без надоедливых анимаций (анимации в приложении также могут быть остановлены в настройках).
Кроме того, на сайте есть много ошибок, особенно для мобильной версии, так что вы не можете сделать все правильно.
