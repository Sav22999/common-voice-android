<b>Características de la aplicación:</b>

- La interfaz de usuario (UI) es inteligente, clara y moderna
- Inicia sesión en tu cuenta de Common Voice
- Valida clips
- Graba frases
- Reporta frases/clips en las secciones correspondientes
- Soporta múltiples idiomas
- Modo sin conexión (¡continúa contribuyendo también cuando no estás conectado a Internet!)
- Estadísticas de Common Voice (del idioma seleccionado)
- Mayores contribuyentes (del idioma seleccionado)
- Estadísticas de la aplicación (anónimas)
- Voces en línea (del idioma seleccionado)
- Tema oscuro soportado
- Objetivo diario
- Gestos
- Guarda tus grabaciones en tu dispositivo
- Muchas otras características
- La experiencia de la aplicación es totalmente personalizable en Ajustes

<b>What is the Common Voice project?</b>

Common Voice es la iniciativa de Mozilla para ayudar a enseñar a las máquinas cómo habla la gente real.
La voz es natural, la voz es humana. Por eso estamos entusiasmados con la creación de tecnología de voz utilizable para nuestras máquinas. Pero para crear sistemas de voz, los desarrolladores necesitan una gran cantidad de datos de voz.
La mayoría de los datos utilizados por las grandes empresas no están disponibles para la mayoría de las personas. Creemos que esto reprime la innovación. Así que hemos lanzado Common Voice, un proyecto para ayudar a que el reconocimiento de voz sea abierto y accesible para todos.
Ahora puedes donar tu voz para ayudarnos a construir una base de datos de voz de código abierto que cualquiera pueda utilizar para crear aplicaciones innovadoras para dispositivos y la web. Lee una frase para ayudar a las máquinas a aprender cómo hablan las personas reales. Comprueba el trabajo de otros colaboradores para mejorar la calidad. ¡Es así de simple!
You can find more details on the official website <a href="https://commonvoice.mozilla.org/">https://commonvoice.mozilla.org/</a>
</p>
<p>
<b>¿Por qué deberías usar la aplicación en lugar de la página web en dispositivos móviles?</b>

The app is developed specifically for smartphones so it's native. Tiene muchas características que el sitio web no le proporciona, como tema oscuro, modo sin conexión, personalización, gestos, etc.
La aplicación es más ligera que la página web, sin animaciones molestas (y las animaciones en la aplicación también se pueden detener en Ajustes).
Además, el sitio web tiene muchos errores, especialmente para la versión móvil, por lo que no puedes hacer todo correctamente.
</p>
